## Описание (что изучили)

1. **Циклы в JavaScript**
   - Использование циклов `for`, `while`, `do...while`.
   - Применение операторов `break` и `continue`.
   - Работа с вложенными циклами.

2. **Функции в JavaScript**
   - Объявление функций (классический способ и стрелочные функции).
   - Параметры функций и значения по умолчанию.
   - Функции как значения и замыкания.

3. **Массивы в JavaScript**
   - Создание и доступ к элементам массива.
   - Добавление и удаление элементов (`push`, `pop`).
   - Перебор массивов (`for`, `for...of`).
   - Поиск элементов (`indexOf`, `includes`).

4. **Объекты в JavaScript**
   - Создание объектов (object literal).
   - Доступ к свойствам через точку и квадратные скобки.
   - Добавление и удаление свойств.
   - Объекты с методами и перебор свойств (`for...in`).
   - Вложенные объекты и массивы.
